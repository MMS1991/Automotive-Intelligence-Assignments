# Speed vs Time Analysis Using Python

## Assignment Description
A journey of more than 10 km (or 30 minutes) was recorded using Google Maps with screen recording enabled.
Speed values were extracted at regular time intervals and analyzed using Python.

## Task
Plot the Speed vs Time graph for the recorded journey.

## Tools Used
- Python 3
- matplotlib
- numpy
- Google Maps (for journey recording)

## How to Run
1. Install required libraries:
   pip install matplotlib numpy

2. Run the program:
   python speed_time.py

## Output
A Speed vs Time graph showing speed variation during the journey.

## Additional Work
Acceleration vs Time graph is also included as an additional analysis.
